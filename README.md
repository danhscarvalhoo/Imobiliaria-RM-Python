
# Sistema de Orçamento Imobiliário - R.M
[cite_start]Este projeto automatiza a geração de orçamentos para locação de imóveis (Casas, Apartamentos e Estúdios)[cite: 11, 12].

## Funcionalidades
- [cite_start]Cálculo de mensalidade com base no tipo de imóvel e opcionais (quartos, garagem)[cite: 16].
- [cite_start]Desconto de 5% para apartamentos sem crianças[cite: 28].
- [cite_start]Cálculo de parcelamento do contrato imobiliário (R$ 2.000,00 em 5x)[cite: 20, 29].
- [cite_start]Exportação de cronograma de 12 meses em arquivo .csv[cite: 30].

## Tecnologias
- Python 3.x
- [cite_start]Paradigma de Orientação a Objetos (POO) [cite: 76]
