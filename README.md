
# Sistema de Orçamento Imobiliário - R.M
Este projeto automatiza a geração de orçamentos para locação de imóveis (Casas, Apartamentos e Estúdios).

## Funcionalidades
- Cálculo de mensalidade com base no tipo de imóvel e opcionais (quartos, garagem).
- Desconto de 5% para apartamentos sem crianças.
- Cálculo de parcelamento do contrato imobiliário (R$ 2.000,00 em 5x).
- Exportação de cronograma de 12 meses em arquivo .csv.

## Tecnologias
- Python 3.x

- Paradigma de Orientação a Objetos (POO)
